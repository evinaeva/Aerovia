# Style LoRA — RETIRED

> **This tooling is retired and no longer used.** It was an SDXL LoRA pipeline that
> fixed an *alternate* art look as a trainable style and generated new transparent
> (RGBA) sprites in it. The game now ships a **single look (neon)** — that alternate
> look and its sprite folder were removed, so this generation pipeline has no target.

If you ever need bespoke neon sprites, see the look's brief and sourcing notes:

- [`../../docs/design/skins/neon/BRIEF.md`](../../docs/design/skins/neon/BRIEF.md)
- [`../../docs/design/skins/neon/SOURCING.md`](../../docs/design/skins/neon/SOURCING.md)

The scripts under this folder (dataset prep, RunPod training, ComfyUI generation, QA)
are kept only as a historical reference and are not wired into the project.
