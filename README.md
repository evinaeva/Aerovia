# Tower

Мобильная игра-«диспетчер»: веди самолёты пальцем к посадочной полосе нужного
цвета, не допуская столкновений в воздухе. Ощущение — спокойный поток, лёгкая
медитация.

HTML5 Canvas + ванильный JS, без зависимостей.

## Запуск

Открыть `index.html` в браузере. Для теста на телефоне удобнее поднять
локальный сервер:

```sh
python3 -m http.server 8000
# затем открыть http://<ip-компьютера>:8000 на телефоне в той же сети
```

Или включить **GitHub Pages** (Settings → Pages → ветка `main`, корень) —
игра откроется по ссылке вида `https://<user>.github.io/<repo>/`.

## Структура

- `index.html` — игра (точка входа, она же стартовый прототип).
- `docs/tower_project_brief.md` — бриф проекта: цель, роли, бэклог, план до Google Play.
