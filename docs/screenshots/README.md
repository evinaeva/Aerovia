# Галерея версий PlaneFlow

Скриншоты главного экрана, геймплея и меню паузы по версиям — чтобы видеть прогресс. Новые сверху.

> ⏳ **Галерея ещё не сгенерирована.** Картинки снимает браузер (Chromium), а в облачной
> песочнице Claude Code он недоступен. Запусти генерацию там, где браузер есть:
>
> - **GitHub Actions** (рекомендуется): вкладка **Actions → «Screenshots» → Run workflow**
>   на нужной ветке. Workflow пройдёт по всем версиям из git-истории, снимет по 3 экрана,
>   соберёт эту страницу и закоммитит картинки обратно.
> - **Локально:** `npm install && npx playwright install chromium`, затем
>   `npm run screenshots -- --all && npm run gallery`.
>
> Текстовая история версий — в [`CHANGELOG.md`](../../CHANGELOG.md).

После прогона здесь появится по строке на версию (0.7 → текущая) с тремя картинками:
**Главный экран · Геймплей · Пауза**.
