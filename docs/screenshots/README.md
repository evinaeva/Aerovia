# Галерея версий PlaneFlow

Скриншоты главного экрана, геймплея и меню паузы по версиям — чтобы видеть прогресс. Новые сверху.

> Картинки генерируются автоматически: workflow **«Screenshots»** (`.github/workflows/screenshots.yml`)
> прогоняет игру каждой версии в Chromium. Локально — `npm run screenshots -- --all && npm run gallery`.
> Текстовая история — в [`CHANGELOG.md`](../../CHANGELOG.md).
