## 4. Подход (красиво + не жрёт батарею)
Ключ: свечение **уже запечено** в PNG → в кадре НЕ используем `shadowBlur`/живые градиенты (это
главный пожиратель батареи). Тяжёлое рисуем один раз и кэшируем картинкой, дальше — дешёвый блит.
- Растр на основе её PNG, через **промоут per-zone спрайт-пайплайна** в реальный рендер.
- **Фон** — статичный нижний слой, рисуется один раз, блит без перерисовки каждый кадр.
- **ВПП** — спрайт `vpp`, блит по координатам полос (2 шт. справа).
- **Ангары** — рамка как **9-slice** (тянется под размер бэя без искажений) + иконка сервиса по типу.
- **Самолёт** — растеризуем один раз на нужный размер, поворот+блит; типы — **tint** одного спрайта
  (или отдельные ливреи, если оператор даст).
- **HUD** — хром-рамки из WOW-бара (по отсекам), сердца/деньги/таймер рисуем живьём поверх.
- **Батарея:** статический фон отдельным слоем; нет `shadowBlur` в кадре; FPS-троттлинг /
  render-on-demand на статичной сцене; правый размер ассетов под ландшафт-DPI (~2x), оптимизация веса.
- **Ландшафт:** фикс горизонтальной ориентации; согласовать с `ui`-масштабом и вводом
  (рисование маршрута — см. ветку `fix/mobile-taxi-dock`).

1. Шрифт для цифр HUD

Точный TTF/OTF из картинки я не могу “вытащить”, потому что это AI-generated raster image, там нет настоящего шрифтового слоя.

Лучший вариант для повторения стиля

Orbitron SemiBold / Bold

Он ближе всего к цифрам на HUD: футуристичный, геометричный, хорошо читается в UI. Есть на Google Fonts. 

CSS:

font-family: "Orbitron", system-ui, sans-serif;
font-weight: 600;
letter-spacing: 0.04em;

Для эффекта как на картинке:

.hud-number {
  font-family: "Orbitron", system-ui, sans-serif;
  font-weight: 600;
  letter-spacing: 0.04em;
  color: #27E6FF;
  text-shadow:
    0 0 6px rgba(39, 230, 255, 0.85),
    0 0 14px rgba(39, 230, 255, 0.45);
}

Альтернатива

Oxanium SemiBold / Bold

Чуть мягче и более “игровой”, тоже подходит для sci-fi HUD. Есть на Google Fonts. 

font-family: "Oxanium", system-ui, sans-serif;
font-weight: 600;

Для Aerovia я бы выбрала:

Primary HUD font: Orbitron SemiBold
Fallback: Oxanium SemiBold

Файл шрифта я не прикладываю. Нужно скачать/подключить шрифт легально из источника и проверить лицензию перед включением в игру. Google Fonts обычно используется как библиотека свободных/open-source шрифтов, многие из них распространяются под SIL Open Font License.

3. Палитра HEX

Ориентировочная палитра по выбранному экрану:

Назначение	HEX	Комментарий

Тёмный основной фон	#061426	почти чёрно-синий фон UI
Декоративный sci-fi фон	#0A1D34	фон вокруг апрона / подложка
Апрон / синяя плитка	#1D385D	основная поверхность поля
Синий неон рамок	#00CFFF	рамки, подсветка, контуры
Голубой HUD-текст	#27E6FF	деньги, таймер, цифры
Маршрутная линия	#16BFEF	лучше чуть мягче, чем рамки
Оранжевый акцент	#FF9B1A	сервис/топливо/предупреждения
Красный сердечек	#FF4A4A	жизни


CSS-переменные:

:root {
  --pf-bg-dark: #061426;
  --pf-bg-panel: #0A1D34;
  --pf-apron: #1D385D;

  --pf-neon-blue: #00CFFF;
  --pf-text-cyan: #27E6FF;
  --pf-route-line: #16BFEF;

  --pf-accent-orange: #FF9B1A;
  --pf-heart-red: #FF4A4A;
}

Для траекторий я бы не использовала чистый #00CFFF, потому что он совпадает с рамками. Лучше маршрутную линию делать чуть светлее/мягче или с прозрачностью:

--pf-route-line: rgba(22, 191, 239, 0.85);

Так игрок будет лучше отличать рамку поля от нарисованной траектории.
