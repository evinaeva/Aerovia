# Арт-бэклог — что придумать и отрисовать

Статусы: ✅ есть (можно улучшать) · 🟡 частично · ⬜ нужно · 🏪 для публикации.
Ведём вместе с остальной докой: отмечаем по мере готовности.

## 1. Бренд / идентичность
- ⬜ **Название** (сейчас рабочее «Tower»; нужно финальное, уникальное)
- ⬜ **Логотип / вордмарк**
- 🟡 **Палитра** (рабочая есть: amber/teal/rose/gold/phosphor + тёмный апрон — утвердить)
- 🟡 **Шрифты** (моноширинный + Georgia — утвердить)
- 🏪 **Иконка приложения** (Google Play / iOS)

## 2. Игровые объекты (на поле)
- ✅ **Самолёты:** обычный (белый), вип (золотой); состояния «выбран», «терпение на исходе»
- ✅ **Иконки услуг:** ремонт, топливо, посадка людей, вылет (утвердить финальный стиль)
- ✅ **Боксы:** рамка, открытый/закрытый (замок), по типу, уровень апгрейда, прогресс, «занято»
- ✅ **ВПП:** разметка, осевая, бортовые огни, порог
- ✅ **Апрон/поле:** тармак, рама, центральный терминал, трава, вода
- ✅ **Траектория** маршрута (обычная/выделенная)

## 3. Эффекты (анимации)
- ✅ **Крушение** (взрыв/искры)
- ⬜ **Посадка** (касание, дымок шасси)
- ⬜ **Взлёт** (разгон, отрыв)
- ⬜ **Обслуживание в боксе** (разное по типу услуги)
- ⬜ **+Деньги при вылете** (всплывающая цифра)
- 🟡 **Подсветка успеха/ошибки** (есть toast; докрутить)

## 4. HUD / интерфейс
- 🟡 **Жизни** (сейчас ♥ текстом — нужен значок)
- ⬜ **Деньги** (значок валюты/монета)
- ✅ **Таймер** (прямой/обратный), **счётчик метрики + звёзды**
- ✅ **Кнопка паузы**
- ✅ **Цепочка операций** выбранного борта
- 🟡 **Всплывашки (toast):** потеря жизни / уровень пройден (+ нужно «+деньги»)
- 🟡 **Звёзды 0–3** (сейчас текстом ★/☆ — нужен значок)

## 5. Экраны / меню
- 🟡 **Главное меню** (фон/заголовок — базово)
- 🟡 **Выбор уровней** (карточки есть; нужны миниатюры/превью уровня)
- ✅ **Пауза**, ✅ **Конец смены** (итоги + звёзды)
- ✅ **Кнопки** (единый стиль)

## 6. Для публикации
- 🏪 **Иконка приложения** 512×512 + adaptive (Android)
- 🏪 **Скриншоты** (телефон/планшет, с подписями)
- 🏪 **Feature graphic** 1024×500 (Google Play)
- 🏪 **Промо-баннеры / превью-видео**

## 7. Звук и тактиль (не отрисовка, но в общий бэклог ассетов)
- ⬜ Приём борта, посадка, обслуживание, взлёт, краш, нажатия, фоновый эмбиент
- ⬜ Вибрация на ключевых событиях

---

**Приоритет «придумать с нуля»:** название + логотип + иконка приложения (без них не
выйти в стор); значок звезды (уже используется). Остальное — улучшение плейсхолдеров.
