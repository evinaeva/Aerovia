# Арт-бэклог — что придумать и отрисовать

Статусы: ✅ есть (можно улучшать) · 🟡 частично · ⬜ нужно · 🏪 для публикации.
Ведём вместе с остальной докой: отмечаем по мере готовности.

> **2026-06-12 — интегрирована дизайн-система PlaneFlow (cozy / stylized-flat).**
> Канонические спрайты: `assets/sprites/planeflow-*.svg` (бренд / самолёты / поле /
> HUD / эффекты), спека — `assets/sprites/README.md`. В игре рисуются через
> спрайт-атлас (`SPRITES`, флаг `COZY`) с процедурным фолбэком.

## 1. Бренд / идентичность
- ✅ **Название** — **PlaneFlow** (утверждено 2026-06-12; было рабочее «Tower»)
- ✅ **Логотип / вордмарк** — `mark` / `wordmark` / `wordmark-lockup`; локап на стартовом экране
- ✅ **Палитра** — cozy-токены утверждены (navy/purple база, cream текст, акценты услуг); спрайты перекрашиваемы на лету через `var(--token, #hex)` + `SPRITES.setTheme()` (см. `assets/sprites/README.md`)
- 🟡 **Шрифты** — display-стек в вордмарке (Fredoka — substitution), в продукте mono; финализировать при пакетировании
- ✅ **Иконка приложения** — `assets/icon/`: 512 full-bleed + Android adaptive (bg/fg); подключена как favicon/apple-touch-icon

## 2. Игровые объекты (на поле)
- ✅ **Самолёты:** обычный (cream), вип (золотой), аварийный, медицинский; кольца «выбран» / «терпение» поверх
- ✅ **Иконки услуг:** cozy-чипы (Lucide-стиль) — над бортом, в боксах, в цепочке операций
- ✅ **Боксы:** ворота к полю (процедурно) + cozy-чипы типа, замок, уровень, прогресс, «занято» + живая анимация обслуживания
- ✅ **ВПП:** разметка, осевая, бортовые огни (amber-glow), порог; состояния занята/закрыта
- ✅ **Апрон/поле:** бесшовные тайлы (тармак-плиты, трава, вода) + песчаный берег с пеной; терминал-спрайт
- ✅ **Траектория** маршрута: зелёная, со стрелкой-наконечником и точкой-целью

## 3. Эффекты (анимации)
- ✅ **Крушение** — `fx-crash`
- ✅ **Посадка** — дымок касания `fx-touchdown`
- ✅ **Взлёт** — спид-лайны `fx-takeoff`
- ✅ **Обслуживание в боксе** — по типу: искры/капли/посадка (`fx-weld/fuel/boarding/droplet`)
- ✅ **+Деньги при вылете** — флоты `+N ₿ / ×N combo / on time!`
- ✅ **Подсветка успеха/ошибки** — `fx-success`/`fx-error` + иконки тостов
- ⬜ **Зимний арт** — снег-тайл, занос, плуг (спрайтов нет; механика `weather` opt-in выключена в кампании)

## 4. HUD / интерфейс
- ✅ **Жизни** — спрайты `heart`/`heart-empty`
- ✅ **Деньги** — монета `coin` + значение золотом
- ✅ **Таймер** (спрайт-часы), **счётчик метрики**
- ✅ **Кнопка паузы** — `pause-btn`
- ✅ **Цепочка операций** выбранного борта — cozy-чипы
- ✅ **Всплывашки (toast)** — единый стиль + иконки `check`/`heart-crack`
- ✅ **Звёзды 0–3** — спрайт (карточки уровней, экраны)

## 5. Экраны / меню
- ✅ **Главное меню** — фон `menu-bg` + вордмарк-локап
- ✅ **Выбор уровней** — карточки по спеке: мини-превью борда, номер, звёзды, рекорд, замок
- ✅ **Пауза**, ✅ **Конец смены** (итоги + звёзды)
- ✅ **Кнопки** — единая система; primary — зелёная по спеке
