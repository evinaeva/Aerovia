# Что изменилось — заметка верстальщику (последняя ревизия)

Кратко, чтобы не перечитывать весь HANDOFF.md. Это правки поверх предыдущей версии макета.

## Боксы (стойла ангара)
- **Точки апгрейда переосмыслены.** Раньше были в цвете услуги — теперь **зелёные = куплено,
  пустые (полые) = доступно**, одинаково для всех услуг. Лежат в **плашке со скруглёнными
  краями**, отцентрованы на стене, между иконкой услуги (слева) и кнопкой апгрейда (справа) —
  **никогда не перекрывают иконки**. Диапазон: **минимум 1, максимум 4** точки.
- **Иконка услуги увеличена ещё на 20%** (бейдж 44px, глиф 27px) и **полностью непрозрачная**
  (фон `#0c1736`) — сквозь неё не видно стену.
- **Замок и сумма** у закрытого бокса — на **непрозрачной подложке** (`#0a1326`), ангар сквозь
  них не просвечивает.
- Стойла **уменьшены** (под 1 самолёт), **стены между боксами толще** (16px).
- При **полной прокачке** кнопка апгрейда `↑` **исчезает** — остаются только зелёные точки.
- Иконка `board` теперь **пассажиры** (две фигуры), а не саквояж (раньше путалась с замком).
- Надпись «TAP TO OPEN» убрана.

## Борт (самолёт)
- Показывается **только одна иконка за раз — текущая нужда**. Пока борт **в боксе — иконки нет**.
- Иконка нужды **слегка перекрывает** самолётик (сидит на носу), а не висит в стороне.
- **Таймеры терпения с самолёта убраны.**
- Статус «READY» заменён **иконкой взлёта**.
- Масштаб: в ангаре == на земле; **в воздухе самолёт крупнее**.

## Поле / апрон
- **Декор-радар (дуги, вращающаяся антенна) и дирижабль — удалены.** Остались вышка, силуэт
  города, звёзды, маяки, огни подхода, перронная техника.
- Ангары **прижаты к верх/низ кромкам апрона**, **3 ВПП вертикально симметричны** (равные
  отступы сверху/снизу), между ВПП и стеной ангара выдержан зазор.

## HUD
- Время матча — **просто число** справа (без иконки часов), пауза отодвинута от угла.
- Инфо-бар борта (вариант **Bar**) — **под верхним HUD**, левый край **выровнен по левому краю
  апрона**.

## Что НЕ для прода
Верхние пикеры (стиль ангара, вариант инфо) и нижние вкладки **Boxes / Borts / Tokens** —
только референс. Дефолт к выпуску: ангар **Sawtooth** + инфо **Bar**.
